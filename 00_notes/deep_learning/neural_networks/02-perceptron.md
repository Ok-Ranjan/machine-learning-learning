# Perceptron (Single Neuron Model)

The **Perceptron** is the simplest neural network model.

It consists of a single neuron used mainly for **binary classification**.

Example:

Loan approval prediction.

Inputs:
- Income
- Credit score
- Debt ratio

Output:

Approved → 1  
Rejected → 0
