# Multi-Layer Neural Network

Modern neural networks contain multiple hidden layers.

Input Layer
↓
Hidden Layer 1
↓
Hidden Layer 2
↓
Hidden Layer 3
↓
Output Layer

Depth of the network = number of hidden layers.

This is why it is called **Deep Learning**.

---

## Output Layer Design

### Regression

Output neurons = 1

Example:

House price prediction.

---

### Binary Classification

Output neurons = 1

Activation = Sigmoid

---

### Multi-Class Classification

Output neurons = number of classes

Example:

3 classes → 3 output neurons

Activation = Softmax
